常見問題

設定改了但頁面沒有更新

  • 打開 chrome://extensions
  • 找到 邦尼逃 bunnytao
  • 點擊 Reload
  • 回原頁面重新整理

某個網站頁面跑版

建議回報以下資訊：

  • 問題頁面的網址
  • 使用的模式（淺色模式或深色模式）
  • 問題截圖
  • 問題區塊的 HTML 或 selector（如果方便）

快速切換頁面沒有反應

  • 確認快捷鍵是否與其他擴充功能衝突
  • 可到 chrome://extensions/shortcuts 調整快捷鍵
