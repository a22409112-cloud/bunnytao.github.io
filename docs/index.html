產品定位

這是一款以「低干擾顯示」為核心的瀏覽輔助工具。它不會重做網站架構，而是透過樣式調整與輔助功能，讓畫面更一致、資訊更集中。

核心功能

低干擾顯示

提供淺色模式與深色模式，降低高辨識色彩與視覺噪音。

中性詞彙顯示

將部分詞彙轉為較中性的顯示方式，讓畫面語氣更一致。

簡化干擾區塊

弱化品牌標誌、浮動元件與非主要資訊區塊，集中閱讀焦點。

收藏與稍後看

將目前頁面加入清單，方便之後集中回看與整理。

快速切換頁面

支援研究筆記頁與行事曆頁模板，便於快速切換工作情境。
