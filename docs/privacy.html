Privacy Policy

邦尼逃 bunnytao 隱私政策

說明本擴充功能會處理哪些資料、如何使用資料，以及使用者可如何管理相關資料。

Last updated: 2026-03-22

1. 本擴充功能的用途

邦尼逃 bunnytao 是一款求職網站瀏覽輔助工具，用途為降低指定求職網站的畫面干擾程度，並提供與瀏覽輔助相關的收藏與快速切換功能。

2. 本擴充功能會處理哪些資料

使用者設定

  • 是否啟用樣式
  • 顯示模式
  • 快速切換頁面模板
  • 關鍵字替換設定
  • 干擾區塊隱藏設定

上述設定儲存在 Chrome storage sync。

收藏與稍後看資料

  • 網站名稱
  • 頁面標題
  • 公司名稱或頁面中可取得的基本資訊
  • 頁面網址
  • 收藏時間
  • 內部識別用 ID

上述資料儲存在 Chrome storage local。

目前分頁資訊

當使用者開啟 popup、儲存頁面或切換功能時，本擴充功能會讀取目前分頁的基本資訊，例如網址、分頁標題與頁面上的基本中介資料。

3. 我們如何使用這些資料

  • 套用視覺調整與瀏覽輔助功能
  • 顯示目前網站狀態
  • 建立收藏與稍後看清單
  • 快速切換到中性頁面模板

4. 我們不做的事

  • 不會建立使用者帳號
  • 不會要求登入外部服務
  • 不會將資料上傳到外部伺服器
  • 不會將資料出售、交換或分享給第三方
  • 不會將資料用於廣告投放或追蹤分析

5. 權限使用說明

storage

用於儲存使用者設定與收藏資料。

tabs

用於讀取目前分頁資訊、取得使用者主動收藏頁面的基本資料，以及開啟使用者主動操作的頁面。

host permissions

  • https://*.104.com.tw/*
  • https://*.cake.me/*
  • https://*.cakeresume.com/*
  • https://*.yourator.co/*
  • https://*.linkedin.com/*

6. 資料保存與管理

本擴充功能的資料保存在瀏覽器提供的 Chrome storage 中。保存時間取決於使用者是否手動刪除收藏資料、重設擴充功能資料，或移除本擴充功能。

使用者可以在擴充功能 popup 中調整功能設定、刪除收藏資料，或透過移除本擴充功能刪除相關本地資料。

7. 第三方網站

本擴充功能會在使用者主動開啟的第三方網站上運作，但本政策僅適用於本擴充功能本身，不適用於 104、Cake、Yourator、LinkedIn 或其他第三方網站的隱私政策與資料處理方式。
